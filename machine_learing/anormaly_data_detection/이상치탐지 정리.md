### 이상치 탐지(Abnomaly Detection)이란 
 ```
 정상 범주에서 벗어나 있는 모든 상태 또는 물체를 감지하는 것을 말한다.
```

#### 이상치탐지의 대한 분류[학습 데이터에 따른 분류]

```
 지도 학습 방법
 정상 데이터와 비정상 데이터 두 가지 다 Label이 존재하는 경우를 말한다.
 장점: 높은 정확도
 단점: 데이터셋 구축이 힘들고 구축 데이터셋에 대하여 Class-Imbalance 문제를 해결해야함.

  반지도 학습 방법
  정상 데이터 샘플만을 가지고 있는 경우를 말한다. 
  장점: 정상 데이터만으로 학습이 가능하다.
  단점: 지도학습에 비하여 정확도가 낮다.
  
  비지도 학습 방법
  Label 존재하지 않은 상태에서 학습하는 방법이다.
  장점: 레이블링이 필요하지 않아 지도 학습에 비해 데이터셋 구축이 편하다.
  단점: 정확도가 높지 않으며 하이퍼 파라미터에 민감하다.  
```

```
  비정상 정의에 따른 분류
  Novelity Detection - 기존에 없는 새로운 형태의 샘플을 찾아내는 방식을 말한다.
  Outlier Detection - 샘플들과 상관이 없는 데이터를 찾아내는 방식을 말한다.
  
  Outlier Detection 격리 포레스트
  데이터를 트리를 사용하여 게속하여 격리시키는 알고리즘이다. 
  일반적인 데이터에 비해 이상치 데이터는 멀리 떨어져있다는 점을 착안하여 만들었다.
  간단하고 빠른 속도를 보여주나 특정 케이스 격리한 분포도가 겹치는 부분에 이상치가 존재하는 경우나
  데이터들이 다닥다닥 붙어있어 분포도가 데이터 전체를 덮는 경우는 정상적인 탐지가 불가하다.
  
  Auto Encoder
  오토 인코더에서 노이즈 특징에 대하여 학습시키는 방법이다.
  일반적인 데이터가 오토인코더에 들어갔을 경우 input과 output에 차이가 거의 없으나
  비정상 샘플이 들어갈 경우 noise가 제거된 output으로 인하여 input과 output의 loss값이 커지게되는 것으로 탐지한다.
  
  
```
